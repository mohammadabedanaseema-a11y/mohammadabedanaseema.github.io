# Smart Student Information Management System

## Overview
A web-based system to manage student records, attendance, and marks.

## Features
- Admin login
- Add, update, delete student records
- Search students
- Attendance management
- Marks management
- Grade calculation

## Technologies Used
- Python
- Flask
- HTML
- CSS
- SQLite

## Modules
- Admin Login
- Dashboard
- Student Management
- Attendance Module
- Marks Module

## How to Run
1. Install Python
2. Install Flask using `pip install flask`
3. Run `python app.py`
4. Open `http://127.0.0.1:5000`
